welk css framework word er gebruikt?
de css framework die gebruikt word is bootstrap

wat is het product?
Bootstrap is een verzameling hulpmiddelen voor het maken van websites en webtoepassingen. 
Het is gratis en open source. Het bevat sjablonen gebaseerd op HTML en CSS voor typografie, 
formulieren, knoppen, navigatie en andere interfaceonderdelen. Het bevat ook JavaScript-extensies.

hoe het gebruikt word?
bootstrap roep je aan met css daarmeeheb je bepaalde tags die je kan gebruiken, dit zorgt ervoor dat je je website snel, makkelijk en responsive kan opstellen
